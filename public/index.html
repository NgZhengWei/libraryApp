My Library

About This Project

▼

This project is inspired by The Odin Project. This project features Object Manipulation, Local Storage and Firebase.

This application is written using only vanila HTML5, CSS3 and Javascript. No frameworks are used.

How To Use

▼

This app serves as a virtual library for you to keep track of the books that you are reading.

Press the '+' icon on the bottom right to add a new book.

Click on 'read' status to change the status of the book.

Delete button does exactly what you would expect it to.

To save the changes you made to you library, login using your gmail account.
